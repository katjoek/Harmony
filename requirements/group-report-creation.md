# Report creation
    - There should be an item in the main menu "Reports" (in Dutch)
    - When clicking it, the user must be able to
        1. select a group to report
        2. select a which fields from the user should be reported (the full name should be one of them)
        3. select an output format: PDF or Excel
        3. click a Create button
    - The report should contain:
        - a title showing the group's name
        - the timestamp of when the report was created
        - the coordinator's name, if present (with a label)
        - a table for all the members in the selected group, with columns for all the selected fields.
    - Clicking the Create button should create a report in the format the user selected
    - The name of the downloaded file should contain the group name and timestamp.
