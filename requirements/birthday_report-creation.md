# Birthday report creation
    - Allow the user to select to create a birthday report.
    - When selected, the user should select a month.
    - The user should also be able to select PDF or Excel, Portait or Landscape and the fields to add in the report. These also are available to the user when creating a group report. The parts that it has in common with the existing report should be moved to a shared component.
    - The report created for the birthdays should say "Verjaardagen in " appended with the selected month (in Dutch).
    - The people in the report should have their birthday in the user selected month.
    - The people in the report should be ordered by month and then day of the month, ascendingly.
